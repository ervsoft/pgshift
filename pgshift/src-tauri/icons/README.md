# Icon files needed for Tauri bundling

To generate proper icons, use Tauri's icon generation tool:

```bash
pnpm tauri icon path/to/your-icon.png
```

Or create the following icon files manually:
- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)

For development, you can use placeholder icons.
